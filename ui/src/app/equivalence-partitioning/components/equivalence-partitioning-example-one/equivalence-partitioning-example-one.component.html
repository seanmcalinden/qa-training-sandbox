Example 1: Min length name

As a new user

I would like to create a username that is at least 5 characters long

So that I can create a great username

1. When I enter less than 5 characters

A validation message should appear stating: 'A minimum of 5 characters is required'

2. When I enter 5 characters or more

The name is valid

There are 2 partitions we can extract from the acceptance criteria.

Acceptance criteria 1 tells us that anything with less than 5 characters is invalid.

Acceptance criteria 2 tells us that anything with 5 or more characters is valid.

Invalid Valid
0-4 5+
Whether the length of the name is 0, 1, 2, 3 or 4 characters long, the validation will show the username as invalid. For any names that are 5 characters or more, the validation will show the username as valid
